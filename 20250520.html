2次関数のグラフとx軸との共有点

2次関数 \(y = ax^2 + bx + c\) のグラフとx軸との共有点の個数は、対応する2次方程式 \(ax^2 + bx + c = 0\) の実数解の個数と一致します。この関係を利用して、判別式を用いて共有点の個数を調べることができます。

判別式と共有点の個数 📜

📜 ★★★

2次関数 \(y = ax^2 + bx + c\) のグラフとx軸との共有点の個数は、2次方程式 \(ax^2 + bx + c = 0\) の判別式 \(D = b^2-4ac\) の符号によって次のように決まります。

  • \(D > 0\) のとき: 共有点は2個（異なる2点で交わる）
  • \(D = 0\) のとき: 共有点は1個（1点で接する）
  • \(D < 0\) のとき: 共有点は0個（x軸と交わらない）

グラフの形状（下に凸か上に凸か）は \(a\) の符号で決まりますが、共有点の個数自体は判別式 \(D\) のみで判断できます。

➕

補足: グラフの形状と判別式

2次関数のグラフとx軸の位置関係は、\(a\) の符号と \(D\) の符号の組み合わせで以下の6パターンに分類されます。

  • \(a > 0\) (下に凸)
    • \(D > 0\): x軸と2点で交わる
    • \(D = 0\): x軸と1点で接する (頂点がx軸上)
    • \(D < 0\): x軸と交わらない (グラフ全体がx軸より上)
  • \(a < 0\) (上に凸)
    • \(D > 0\): x軸と2点で交わる
    • \(D = 0\): x軸と1点で接する (頂点がx軸上)
    • \(D < 0\): x軸と交わらない (グラフ全体がx軸より下)

グラフがx軸と接する条件 ⭐

⭐ ★★★

2次関数のグラフがx軸に接するとは、共有点が1個である状態を指します。これは、判別式 \(D=0\) と同値です。

この条件は、グラフの頂点がx軸上にあることも意味します。

例題1: 共有点の個数の判別 ✍️ ★☆☆ ★★☆

次の2次関数のグラフとx軸の共有点の個数を求めなさい。

\(y = x^2 + 2x - 1\)

解答・解説

2次方程式 \(x^2 + 2x - 1 = 0\) の判別式 \(D\) を計算します。

\(a=1, b=2, c=-1\) なので、

\(D = b^2 - 4ac = (2)^2 - 4(1)(-1) = 4 - (-4) = 4 + 4 = 8\)

\(D = 8 > 0\) なので、グラフとx軸の共有点は2個です。

答え: 2個

割り算のアルゴリズムの再確認 📖

📖 ★★★

整式 \(A\) を整式 \(B\) (\(B \neq 0\)) で割ったときの商を \(Q\)、余りを \(R\) とすると、次の関係式が成り立ちます。

\[ A = BQ + R \]

このとき、(余り \(R\) の次数) ＜ (割る式 \(B\) の次数) となるか、または \(R=0\) (割り切れる場合) です。

筆算の手順は以下の通りです。

  1. 割られる式 \(A\) と割る式 \(B\) を降べきの順に整理します。
  2. \(A\) の最高次の項を \(B\) の最高次の項で割り、商の最初の項 \(Q_1\) を求めます。
  3. \(A\) から \(B \times Q_1\) を引きます。
  4. 得られた余りの式の最高次の項を \(B\) の最高次の項で割り、商の次の項 \(Q_2\) を求めます。
  5. これを、余りの次数が \(B\) の次数より小さくなるまで繰り返します。

例題2: 2次式で割る割り算 ✍️ ★★☆ ★★★

整式 \(A = 4x^3 + 3x + 9\) を整式 \(B = 2x+1\) で割りなさい。（教科書 例題の応用）

解答・解説

割られる式 \(A\) には \(x^2\) の項がないため、\(4x^3 + 0x^2 + 3x + 9\) として筆算を行います。


           2x^2  -  x   + 2
        ____________________
2x+1 | 4x^3 + 0x^2 + 3x + 9
       -(4x^3 + 2x^2)
       ____________________
              -2x^2 + 3x
            -(-2x^2 -  x)
            ____________
                     4x + 9
                   -(4x + 2)
                   _________
                           7

計算手順の解説:

  1. \(A=4x^3+0x^2+3x+9\), \(B=2x+1\)。
  2. \(4x^3 \div 2x = 2x^2\)。商の最初の項は \(2x^2\)。
  3. \(2x^2(2x+1) = 4x^3+2x^2\)。\(A\) から引くと、\((4x^3+0x^2) - (4x^3+2x^2) = -2x^2\)。
  4. 次の項 \(3x\) を下ろし、\(-2x^2+3x\)。
  5. \(-2x^2 \div 2x = -x\)。商の次の項は \(-x\)。
  6. \(-x(2x+1) = -2x^2-x\)。\((-2x^2+3x)\) から引くと、\((-2x^2+3x) - (-2x^2-x) = 4x\)。
  7. 次の項 \(9\) を下ろし、\(4x+9\)。
  8. \(4x \div 2x = 2\)。商の次の項は \(+2\)。
  9. \(2(2x+1) = 4x+2\)。\((4x+9)\) から引くと、\((4x+9) - (4x+2) = 7\)。
  10. 余り \(7\) (0次) は割る式 \(2x+1\) (1次) より次数が小さいので終了。

したがって、商は \(2x^2-x+2\)、余りは \(7\) です。

関係式で表すと、\(4x^3 + 3x + 9 = (2x+1)(2x^2-x+2) + 7\)

答え: 商 \(2x^2-x+2\), 余り \(7\)

💡

Tips: 割り算における次数のない項の扱い

★★★ 割られる式や割る式に、ある次数の項が存在しない場合（係数が0の場合）、筆算を行う際にはその部分のスペースを空けるか、係数0で項を補って書くと計算ミスを防ぎやすくなります。

例: \(A = x^3 - 1\) を \(B = x-1\) で割る場合

\(A = x^3 + 0x^2 + 0x - 1\) と考えて筆算します。


          x^2  +  x   + 1
        _________________
x-1 | x^3 + 0x^2 + 0x - 1
      -(x^3 -  x^2)
      _________________
             x^2 + 0x
           -(x^2 -  x)
           ___________
                   x - 1
                 -(x - 1)
                 _________
                         0

この例では、商が \(x^2+x+1\)、余りが \(0\) (割り切れる) となります。これは因数分解の公式 \(x^3-1 = (x-1)(x^2+x+1)\) に対応しています。

練習問題1: 整式の割り算 ✏️ ★★☆ ★★★

整式 \(A = 2x^3 - x^2 - 4\) を整式 \(B = x^2+x-1\) で割りなさい。

解答・解説

割られる式 \(A\) には \(x\) の項がないため、\(2x^3 - x^2 + 0x - 4\) として筆算を行います。


               2x   - 3
            _________________
x^2+x-1 | 2x^3 -  x^2 + 0x - 4
          -(2x^3 + 2x^2 - 2x)
          _________________
                 -3x^2 + 2x - 4
               -(-3x^2 - 3x + 3)
               _________________
                        5x - 7

したがって、商は \(2x-3\)、余りは \(5x-7\) です。

関係式で表すと、\(2x^3 - x^2 - 4 = (x^2+x-1)(2x-3) + (5x-7)\)

答え: 商 \(2x-3\), 余り \(5x-7\)

理解度チェッククイズ 1

  1. 2次関数 \(y = -x^2 + 4x - 4\) のグラフとx軸の共有点の個数はいくつですか？

    1. 0個
    2. 1個
    3. 2個
    答えと解説

    答え: B. 1個

    解説: 判別式 \(D = b^2-4ac = 4^2 - 4(-1)(-4) = 16 - 16 = 0\)。\(D=0\) なので共有点は1個（接する）。

    別解: \(y = -(x^2-4x+4) = -(x-2)^2\)。頂点が(2,0)なのでx軸と1点で接する。

  2. 整式 \(A(x)\) を \(B(x)=x-2\) で割ったときの商が \(Q(x)=x+1\)、余りが \(R(x)=3\) であるとき、\(A(x)\) は何ですか？

    1. \(x^2 - x + 1\)
    2. \(x^2 + x - 2\)
    3. \(x^2 - x - 5\)
    4. \(x^2 - 3x + 5\)
    答えと解説

    答え: A. \(x^2 - x + 1\)

    解説: \(A(x) = B(x)Q(x) + R(x)\) の関係より、

    \(A(x) = (x-2)(x+1) + 3 = (x^2 + x - 2x - 2) + 3 = x^2 - x - 2 + 3 = x^2 - x + 1\)。

🔖

本日のまとめ

  • 2次関数 \(y=ax^2+bx+c\) のグラフとx軸の共有点の個数は、判別式 \(D=b^2-4ac\) の符号で決まります。 (\(D>0\) で2個, \(D=0\) で1個(接する), \(D<0\) で0個)
  • 整式の割り算は、\(A=BQ+R\) (余りの次数 ＜ 割る式の次数) の関係に基づき、筆算で行います。
  • 筆算の際は、降べきの順に整理し、ない次数の項はスペースを空けるか0で補い、引き算の符号に注意します。
  • 「複素数の範囲で因数分解しなさい」と言われたら、解の公式を使って解を求め、\(a(x-\alpha)(x-\beta)\) の形にします。
  • 「2つの解 \(\alpha, \beta\) を持つ2次方程式を作れ」と言われたら、\(x^2-(\alpha+\beta)x+\alpha\beta=0\) を利用します。
✏️

標準的な練習問題（追加生成）

  1. 2次関数 \(y = 2x^2 - 4x + k\) のグラフがx軸と接するとき、定数 \(k\) の値を求めなさい。 ★★☆ ★★☆
    解答・解説

    グラフがx軸と接するための条件は、判別式 \(D=0\)。

    \(a=2, b=-4, c=k\)。

    \(D = (-4)^2 - 4(2)(k) = 16 - 8k\)。

    \(D=0\) より、\(16-8k=0 \Rightarrow 8k=16 \Rightarrow k=2\)。

    答え: \(k=2\)

  2. 整式 \(3x^3 + 2x^2 - 7x - 5\) を \(x^2-x+1\) で割ったときの商と余りを求めなさい。 ★★☆ ★★★
    解答・解説 
    
           3x   + 5
        ___________________
x^2-x+1 | 3x^3 + 2x^2 - 7x - 5
          -(3x^3 - 3x^2 + 3x)
          ___________________
                 5x^2 - 10x - 5
               -(5x^2 -  5x + 5)
               _________________
                       -5x - 10

    商: \(3x+5\), 余り: \(-5x-10\)

発展的な練習問題（追加生成）

  1. 2次関数 \(y = x^2 + (m-1)x + m^2 - 3\) のグラフがx軸と共有点を持たないとき、定数 \(m\) の値の範囲を求めなさい。 ★★★ ★★☆
    解答・解説

    グラフがx軸と共有点を持たないための条件は、判別式 \(D < 0\)。

    \(a=1, b=m-1, c=m^2-3\)。

    \(D = (m-1)^2 - 4(1)(m^2-3)\)

    \(= m^2 - 2m + 1 - 4m^2 + 12\)

    \(= -3m^2 - 2m + 13\)

    \(D < 0\) より、\(-3m^2 - 2m + 13 < 0\)。

    両辺に-1を掛けて不等号の向きを変えると、\(3m^2 + 2m - 13 > 0\)。

    2次方程式 \(3m^2 + 2m - 13 = 0\) の解を求めると、解の公式より

    \(m = \frac{-2 \pm \sqrt{2^2 - 4(3)(-13)}}{2(3)} = \frac{-2 \pm \sqrt{4 + 156}}{6} = \frac{-2 \pm \sqrt{160}}{6} = \frac{-2 \pm 4\sqrt{10}}{6} = \frac{-1 \pm 2\sqrt{10}}{3}\)

    よって、\(3m^2 + 2m - 13 > 0\) の解は、\(m < \frac{-1-2\sqrt{10}}{3}\) または \(m > \frac{-1+2\sqrt{10}}{3}\)。

    答え: \(m < \frac{-1-2\sqrt{10}}{3}, m > \frac{-1+2\sqrt{10}}{3}\)

  2. 整式 \(A = x^3 + ax^2 - x + b\) を \(x-1\) で割ると余りが \(3\) であり、\(x+2\) で割ると余りが \(-3\) である。このとき、定数 \(a, b\) の値を求めなさい。 ★★★ ★★★
    解答・解説

    剰余の定理より、

    \(A(x)\) を \(x-1\) で割った余りが3であるから、\(A(1)=3\)。

    \(A(1) = 1^3 + a(1^2) - 1 + b = 1 + a - 1 + b = a+b\)。

    よって、\(a+b=3 \quad \cdots ①\)

    \(A(x)\) を \(x+2\) で割った余りが-3であるから、\(A(-2)=-3\)。

    \(A(-2) = (-2)^3 + a((-2)^2) - (-2) + b = -8 + 4a + 2 + b = 4a+b-6\)。

    よって、\(4a+b-6=-3 \Rightarrow 4a+b=3 \quad \cdots ②\)

    ② - ① より、\(3a = 0 \Rightarrow a=0\)。

    これを①に代入して、\(0+b=3 \Rightarrow b=3\)。

    答え: \(a=0, b=3\)
